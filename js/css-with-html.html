#2 Why use CSS "COLOR NAMES" over "HEX VALUES" and vice versa?


- HEX VALUES , give us an option to pick specific shades of a color. This gives us a wide variety of color options, this is also the more professional and preferred method.



-Using a Color Name can be an easy quick method if the developer can NOT remember the hex code for the color they want, or simply just need a base color. It should be noted that specifying Colors By Name, is NOT the preferred method and is NOT something you will see used often.
